 **newlen**/mo\_name/

   mo\_name is type character, default is **-cmo-**

     Changes the length of all memory managed arrays for Mesh Object
 mo\_name to the proper length.

 **EXAMPLES:**

  **cmo/newlen**/mo\_tet2

  **cmo/newlen/-cmo-**

  **cmo/newlen**
